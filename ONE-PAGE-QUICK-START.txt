╔═══════════════════════════════════════════════════════════╗
║                  ⚡ THE 4 MAGIC KEYS ⚡                    ║
╚═══════════════════════════════════════════════════════════╝

  ⌘K     Generate/Edit Code
         → Open file, press ⌘K, type what you want
         → Example: "add error handling"

  ⌘L     Chat with AI
         → Press ⌘L, ask anything
         → Example: "explain this code"

  ⌘⇧P    Command Palette
         → Press ⌘⇧P, type any command
         → Example: "Cline: Open"

  ⌘,     Settings
         → Press ⌘, to open settings
         → Search for anything

╔═══════════════════════════════════════════════════════════╗
║              🎯 MOST COMMON TASKS                         ║
╚═══════════════════════════════════════════════════════════╝

  Generate Code:    ⌘K → type request → Enter
  Fix a Bug:        Select code → ⌘K → "fix this"
  Add Feature:      ⌘K → "add login form"
  Ask Question:     ⌘L → ask anything
  Open Cline:       ⌘⇧P → "Cline: Open"
