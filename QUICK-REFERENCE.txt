Activate:
  cd ~/CursorDocsIndex
  source venv/bin/activate

Commands:
  python docs_cli.py ingest <url>           Ingest documentation
  python docs_cli.py search "query"         Search all docs
  python docs_cli.py lookup "topic"         Quick lookup (for AI)
  python docs_cli.py stats                  Show index stats

Examples:
  python docs_cli.py ingest "https://fastapi.tiangolo.com/"
  python docs_cli.py search "authentication"
  python docs_cli.py lookup "OAuth2 implementation"
