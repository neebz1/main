## 📧 Gmail Consolidation

**Quick Setup (5 minutes):**
1. Pick your main Gmail
2. In other Gmail accounts: Settings → Forwarding → Forward to main Gmail
3. In main Gmail: Settings → Accounts → Add "Send mail as"
4. Done! All emails in one inbox.
