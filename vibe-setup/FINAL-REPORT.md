```bash

# Verify keys loaded

```

- OpenRouter API (get key from: https://openrouter.ai/keys)
- Hugging Face Token (get from: https://huggingface.co/settings/tokens)
- OpenAI API Key (optional)
- Anthropic API Key (optional)

```
Double-click .ttf files to install
```
